###pyinotify
>[Pyinotify](https://github.com/seb-m/pyinotify) 是一款监控文件/文件夹变化的Python模块.它基于linux内核inotify机制.
inotify是一个事件驱动型的通知器.它通过三个系统调用将通知从内核空间调入用户空间.pyinotify结合这些系统调用,提供一个抽象和通用的方式来处理这些功能.
